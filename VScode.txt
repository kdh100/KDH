[VScode Python Extension Recommendation]

1. Language
 - Python by Microsoft
	1) 파이썬 코딩/디버깅 가능
	2) 상호작용 콘솔 제공
 - Pylance
	1) 코드 정보 제공
 - Code Runner
	1) 쉽고 빠르게 코딩/디버깅 가능

2. AI assistant
 - IntelliCode
	1) 현재 작성중인 내용과 이전 커멘드 히스토리를 기반으로 코드 제안

3. Lint/Format/Highlight
 - Pylint
	1) 에러 찾기 기능과 스타일 분석 체크 기능 제공
 - Python Indent
	1) 파이썬 자동 들여쓰기 기능
 - Indent rainbow
	1) 들여쓰기 공간 색깔 표시

4. Docstring
 - Auto Docstring
	1) 도큐먼트 양식(""")을 자동으로 생성해줌
