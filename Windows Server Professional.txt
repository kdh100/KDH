1. 초기구성
	* '서버관리자' 관리 > 속성 > 로그온 시 서버 관리자 자동 시작 안함
	* 네트워크 설정
	* 디스플레이 끄기 기능 중지
	* 화면 해상도 확인
	* IE 보안 강화 구성 사용 안함
	* 암호 사용 기간 제한 없음
	* Windows 업데이트 기능 확인하지 않음
	* 바탕화면 배경 색상 변경
	* VMware Tools 프로그램 설치(재부팅)
	* 컴퓨터 이름 변경(재부팅)
	<추가>
	* 시스템 이벤트 추적기 해제
		- 실행 > gpedit.msc
		- 컴퓨터 구성 > Windows 설정 > 보안 설정 > 로컬 정책 > 보안 옵션
		- 대화형 로그온 ~ 사용할 필요 없음 클릭해서 사용
		- 컴퓨터 구성 > 관리 템플릿 > 시스템
		- 시스템 종료 이벤트 추적기 표시 클릭해서 사용안함
	* 비밀번호 필요없이 로그온 설정
		- 실행 > control userpasswords2
		- 사용자 이름과 암호 ~ 체크해제
		- 패스워드 입력
	* 새로운 계정에 관리자 권한 부여
		- 컴퓨터 관리 > 시스템 도구 > 로컬 사용자 및 그룹 > 사용자
		- 새 사용자 생성
		- 신규 사용자 클릭 > 소속 그룹 > 추가
		- 개체 이름을 입력하십시오 칸에 'Administrators' 추가 후 이름 확인 선택 후 적용
			# 관리자 그룹에 포함시켜 권한 부여
		- 기존 Administrator 계정 사용안함 확인

3. 명령
	<shell>
	* shell:startup : 시작프로그램 폴더
	* shutdown $OPTION : 시스템 종료
		- -f : 강제종료
		- -s : 종료
		- -r : 재시작
		- -i : GUI 인터페이스 보여줌
		- -t ss : 시간 설정(default: 30s)
		- -a : 중지
		- -c : 코멘트
	<tool>
	* mstsc : 원격 데스크톱 연결
	* mrt : 악성 소프트웨어 제거
	* dfrgui : 디스크 조각모음
	* mspaint : 그림판
	* notepad : 메모장
	* calc : 계산기
	* taskmgr : 작업 관리자(= [Ctrl] + [Alt] + [Delete])
	<.msc; 관리도구>
	* servermanager : 서버 관리자
	* msconfig : 시스템 구성
	* gpedit.msc : 로컬 그룹 정책 편집기
	* lusrmgr.msc : 로컬 사용자 및 그룹
	* services.msc : 서비스
	* devmgmt.msc : 장치 관리자
	* compmgmt.msc : 컴퓨터 관리
	* diskmgmt.msc : 디스크 관리
	* fsmgmt.msc : 공유 폴더 관리
	* eventnwr.msc : 이벤트 뷰어
	* secpol.msc : 로컬 보안 정책
	* certmgr.msc : 인증서 관리
	* wbadmin.msc : 서버 백업 관리자
	<control>
	* contrl : 제어판
	* control userpasswords2 : 사용자 계정 접속 설정
	* firewall.cpl : 방화벽
	* ncpa.cpl : 네트워크 연결
	* inetcpl.cpl : 인터넷 속성
	* appwiz.cpl : 프로그램 및 기능
4. 실습
	* administrator 계정 복구
		- Windows CD 로 부팅
		- 설치 초기 화면에서 컴퓨터 복구 선택
		- 옵션 선택 화면에서 문제 해결 선택
		- Command Prompt
			# D:
			# cd Windows\System32
			# move Utilman.exe Utilman2.exe
			# copy cmd.exe Utilman.exe
		- OS 재부팅
		- 로그인 창 화면에서 접근 > Command Prompt
		- C:\Windows\System32> net user Administrator /active:yes
		- C:\Windows\System32> net user Administrator *
		- 새로운 암호 입력
		- OS 재부팅

		- 기존 폴더에서 끌어와서 바로가기 생성, 신규 파일 생성
	* 원격 데스크톱 연결

		<Client>
		- 원격 데스크톱 연결
	* 색인 기능 활성화
		- 서버 관리자 실행
		- Windows Search 서비스 기능 추가
		- 제어판 > '색인' 검색 > 색인 옵션
		- 기능 활성화시킬 드라이브(C:, D:...) 선택
	* 파일 확장자명 숨기기 설정 해제
		- 파일 탐색기 > 보기 > 옵션
		- 숨김 파일 표시, 알려진 파일 확장명 숨기기 해제
	* 드라이브 내 데이터 복구 불가 상태 만들기
		- 실행 > cmd(관리자권한)
		- cmd 외 모든 창 닫기
		- cipher /w:<드라이브>:
	* 파일/디렉토리 단위의 암호화
		<암호>
		- 객체 선택 > 속성 > 일반 > 고급
		- 데이터 보호를 위해 내용을 암호화 체크
		- 파일만 암호화 선택
		<암호키 백업>
		- 인증서 내보내기 마법사(팝업) 선택
		- 암호 체크 후 설정 > 인증키 저장할 디렉토리 지정 후 저장
		<인증서 import>
		- 실행 > certmgr.msc
		- 개인용 선택 > 모든 작업 > 가져오기
		- 저장했떤 인증키 불러오기
		- 암호 입력 후 저장
	* 암호 재설정 디스크 만들기
		- 제어판 > 사용자 계정 > 사용자 계정
		- 암호 재설정 디스크 만들기
		- 저장 매체 선택(USB, Floppydisk...) 후 생성
	* 사용자 암호 정책
		<암호 정책>
		- 복잡성 체크
		- 최소 길이 : 8글자
		- 암호 사용 기간 : 1개월 ~ 3개월(1분기)
		- 이전 암호 저장 기능 : 4개
		<계정 잠금 정책>
		- 암호 잠금 임계값 : 3~5회
	* 폴더 공유
		<고급 공유 설정>
		- 제어판 > 네트워크 및 인터넷 > 네트워크 및 공유 센터 > 고급 공유 설정
		- 현재 네트워크 프로필의 네트워크 검색 ON
		- 파일 및 프린터 공유 ON
		<폴더 공유>
		- 폴더 선택 > 속성 > 공유 > 고급 공유

	* OS 비교
		<Linux>						<Windows>

	* RAID
		- 스팬(linear) : 디스크 하나에 차례대로 데이터 저장
			# 복구 불능
		- 스프라이트 : 모든 디스크에 순차적으로 데이터 저장
			# 복구 불능
		- 미러(RAID-1) : 거울처럼 모든 디스크가 동일하게 데이터 저장
			# 복구 가능
			# 디스크 장애 시 제거 후 다시 미러 동기화해서 복구
		- 패리티(RAID-5) : 복구 패리티 데이터를 모든 디스크에 하나씩 넣어놓고 랜덤하게 데이터 저장
			# 복구 가능
			# 디스크 장애 시 볼륨 복구를 통해 장애난 디스크 위치에 자동 실장되어 복구
