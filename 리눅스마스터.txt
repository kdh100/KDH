[소스파일]
    * configure --> make --> make install
        - configure 다음 생성되는 파일은 'Makefile'
    * 설치 방법
        - make : PHP, SAMBA, apache HTTP
        - cmake : MySQL
    * 제거 방법
        - make clean : 이전 컴파일 내용 삭제
        - make disclean : 자동 분류되지 않은 나머지 파일 삭제

[패키지]
    * 온라인 패키지 관리
        - debian : apt-get
        - redhat : yum
        - suse : zypper
    * 오프라인 패키지 관리
        - debian : dpkg
        - redhat : rpm

    * 의존성 무시 강제 삭제

        - GNOME : 리눅스/유닉스, GTK+ 를 이용해 만든 gui 기반 데스크톱 환경
        - KDE : 유닉스, Qt 를 이용해 만든 gui 기반 데스크톱 환경
        - Xfce : 유닉스, 낮은 리소스 점유율과 높은 실행 속도의 데스크톱 환경
        - LXDE : 모든 환경에서 사용 가능한 데스크톱 환경
    * 윈도우 매니저 : 윈도우 배치/표현
    * 디스플레이 매니저 : 사용자 로그인, 세션 관리
    * PDF 확인
        - evince
    * 이미지뷰어
        - eog : 이미지 뷰어
        - GIMP : 이미지 편집/생성/변환
        - ImageMagicK : 비트맵 이미지 뷰어, 생성/편집 가능
        - gwenview
    * 사운드
        - OSS(hannu), ALSA(Jaroslav)
        - totem : 미디어 플레이어
    * 프린트
        - LPRng, CUPS
        - systemV : lp, lpstat, cancel
        - Unix : lpr, lpc, lpq, lprm
    * 스캐너
        - XSANE
    * 파일 관리
        - nautilus(GNOME 기반), dolpin
        - konqueror(KDE 기반)
    * 라이브러리
        - QT, GTK+
        - Xlib : X 서버와 대화 가능한 C 언어로 구현된 라이브러리, 저수준
        - XCB : X 프로토콜 접근 가능, Xlib 보다 향상된 제품

[편집기]
    * pico(리눅스 초기) --> nano(복제판)
    * 가장 먼저 개발된 편집기는 'vi(빌조이)'
    * cat > '.exrc' --> set nu
    * emacs 편집기
        - 리처드 스톨먼(매크로) + 제임스 고슬링(LSIP 언어 기반 기능 추가)
    * emacs 단축기
        - [ctrl + x] - [s] : 현재 작업 버퍼 저장
        - [ctrl + x] - [ctrl + s] : 파일 저장
        - [ctrl + x] - [ctrl + f] : 파일 찾기
        - [ctrl + x] - [ctrl + c] : 종료

[백그라운드]
    * nohup : 백그라운드로 프로세스 유지

[시그널]
    * kill 
        - 1(SIGHUP) : 재시작
        - 2(SIGINT) : 프로세스 종료, Ctrl + C
        - 3(SIGQUIT) : 종료, Ctrl + \
        - 9(SIGKILL) : 강제 종료
        - 15(SIGTERM) : 시그널(정상 종료)
        - 20(SIGSTP) : 프로세스 백그라운드, Ctrl + Z

[스케쥴링]
    * crontab : <분> <시> <일> <월> <일주일> <작업>

[쉘]
    * sh(본), csh(빌조이), tcsh(캔그리어), ksh(콘), bash(폭스,GNU), zsh(스티븐슨)
    * 쉘 히스토리
        - HISTSIZE : 히스토리 저장 개수
        - HISTFILESIZE : 히스토리 파일 크기

[시스템]

[환경변수]
    * '~/.bash_profile' 에 PATH 등록 사용
    * 쉘 변수

[파티션]
    * xfs_repair : 파티션 점검 및 복구
    * 쿼터
        - quota : 현재 사용자 쿼터 정보 확인
        - edquota : 사용자/그룹 쿼터 설정
        - setquota : 쿼터 적용
        - repquota : 쿼터 확인
        - quotaon|off : 쿼터 ON|OFF
        - quotacheck : 쿼터 설정 갱신
    * 쿼터 설정 과정
        - quotacheck --> edquota --> quotaon --> repquota
    * 디스크 쿼터 설정
        - /etc/fstab --> quota, uquota, usrquota
    * RAID
        - RAID-6 : 총 용량 = N - 2 개의 HDD 용량

[우선순위]
    * nice <-10 | --10> bash
    * renice <10 | -10> 1222
    * ps -l : 백그라운드 프로세스의 우선순위 확인

[권한]
    * 기본 파일 권한 : 0644(-rw-r--r--)
    * 특수 권한
        - setUID : 실행 시 파일소유 유저 권한 부여
        - setGID : 실행 시 파일소유 그룹 권한 부여
        - stickybit : (디렉토리) 누구나 파일 생성/수정 가능, 삭제는 root 만 가능

[운영체제]

    * Rocky Linux : RHEL 복제버전의 배포판, CentOS 창립자인 Gregory Kurtzer 주도

[기술동향]
    * GENIVI : 리눅스 커널 기반의 표준화된 자동차용 플랫폼
    * Hadoop : 큰 컴퓨터 클러스터에서 동작하는 분산 응용 프로그램 지원, 자바 소프트웨어
    * KVM : 2005년 개발된 하이퍼바이저, QEMU 라는 CPU 에뮬레이터 사용
    * 라즈베리파이 : 잉글랜드 개발, 신용카드 크기의 싱글 보드 컴퓨터

[네트워크]
    * ip : 라우팅 테이블 정보 출력
    * enp0s3 : CentOS7 버전에서 이더넷카드 장착 시 장치명
    * IANA(인터넷 등장 초기) --> ICANN(국제기구)
    * MAN : 도시 상의 공중 영역 연결을 위한 표준
        - 802.3 이더넷
        - 802.4 토큰 버스
        - 802.5 토큰 링
        - 802.6 MAN(DQDB)
    * X 서버 접근 : xhost 8.8.8.8
    * ISO : OSI 모델 제정
    * 네트워크 설정
        - ifconfig : 네트워크 인터페이스 설정
        - ip : 라우팅 테이블 확인
        - route : 경로 설정
        - ping : 네트워크 연결 테스트
        - traceroute : 네트워크 경로 테스트
        - netstat : 네트워크 연결 상태 확인
        - arp : arp 테이블 확인
        - ethtool : 물리적 연결 확인 + 이더넷 카드 설정 변경
