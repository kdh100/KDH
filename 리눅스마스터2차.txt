[디스크]

    * 기술 
        - Mirroring : 디스크에 에러 발생 시 데이터의 손실을 막기 위해 추가로 하나 이상의 장치에 중복 저장
        - Volume Group : 물리 볼륨을 합쳐서 하나의 물리적 그룹으로 만드는 것
        - Striping : 연속된 데이터를 여러 디스크에 나눠 저장
    
    * LVM (Logical Volume Manager)
        - PE : PV에서 나누어 사용하는 일종의 블록, 4MB로 할당
        - PV : 디스크를 LVM에서 사용할 수 있게 변환하는 작업
        - VG : PV가 모여 만들어진 그룹
        - LV : VG에서 사용자가 필요한 만큼 할당돼서 만들어지는 공간
        - LE : LV가 나누어진 일정한 크기의 블록 , PE와 1:1 대응
 
    * RIAD
        - RAID-5 : 최소 3개의 저장 장치(디스크)를 필요로 하고 1개(약 33.3%)의 패리티를 사용
        - RAID-6 : 최소 4개의 저장 장치를 필요로 하고 2개의 패리티를 사용
            # 스트라이핑 방식, RAID-5 를 확장
